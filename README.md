### auth-apis


### Installation
```shell
$ git clone --depth 1 --single-branch --branch master https://github.com/ganeshrvel/auth-apis.git

$ cd auth-apis
```

```shell
$ yarn
$ yarn start-dev
```

Default username and password: `user` and `password` respectively
